# Brain Games: GCD

Это игра, в которой вам нужно найти наибольший общий делитель (НОД) двух чисел.

## Установка

1. Склонируйте репозиторий или скачайте проект.
2. Установите зависимости: `npm install`.
3. Сделайте файл исполняемым: `chmod +x ./bin/brain-gcd.js`.
4. Запустите игру: `npx brain-gcd`.
